# 用户使用手册

## 1. 访问地址
- 前端入口：http://localhost:3000

## 2. 登录
- 使用测试账号：
  - 用户名：admin
  - 密码：123456
- 登录成功后进入员工管理控制台

## 3. 注册
- 在登录页点击“立即注册”
- 填写用户名、密码、确认密码
- 注册成功后返回登录页

## 4. 员工管理

### 新增员工
- 点击“新增员工”按钮
- 填写姓名、手机号、性别
- 点击“确认添加”保存

### 编辑员工
- 点击员工行右侧“编辑”
- 修改信息并保存

### 删除员工
- 点击“删除”按钮
- 在确认弹窗中点击“确认”

## 5. 退出登录
- 右上角点击“退出”
- 系统返回登录页

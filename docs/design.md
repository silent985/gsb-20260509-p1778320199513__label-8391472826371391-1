# 设计说明

## 1. 设计目标
- 统一视觉风格，满足 SaaS MVP 级质感
- 提升操作流畅度与信息反馈清晰度
- 支持移动端自适应

## 2. 视觉规范
- 风格：现代、轻量、卡片式布局
- 字体：Inter（Google Fonts）
- 色彩：以主色系 + 中性灰为基础
- 交互：按钮 Hover、加载动画、Toast 反馈

## 3. 页面设计

### 登录页
- 渐变背景 + 卡片式登录框
- 输入框带图标增强识别
- 登录态 Loading 动画

### 注册页
- 与登录页一致的卡片风格
- 密码确认校验与错误提示

### 控制台
- 顶部导航 + 用户信息
- 统计卡片（总数/男女比例）
- 员工列表表格 + 操作按钮
- 空状态引导

### 弹窗
- 员工新增/编辑弹窗：统一表单布局
- 删除确认弹窗：自定义 ConfirmModal

## 4. 体验优化
- 表单实时校验与错误提示
- API 异常统一由拦截器处理
- 操作成功/失败均有 Toast 提示

## 5. 响应式策略
- 关键区域采用 flex/grid 响应式布局
- 表格支持横向滚动，保证移动端可用

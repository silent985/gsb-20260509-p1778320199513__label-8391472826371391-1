# 测试说明

## 1. 测试范围
- 登录/注册
- 员工 CRUD
- 权限拦截（未登录访问受限接口）
- UI 交互与提示

## 2. 运行验证（Docker）
```bash
docker compose up --build
```

## 3. 手工测试用例

### 用例 1：登录
- 输入管理员账号：admin / 123456
- 预期：登录成功，进入控制台，显示用户名

### 用例 2：注册
- 新建账号（用户名 3-20 字符，密码 6-20 字符）
- 预期：注册成功，跳转登录页

### 用例 3：新增员工
- 点击“新增员工”，填写姓名、手机号、性别
- 预期：新增成功，列表出现新记录，Toast 提示

### 用例 4：编辑员工
- 点击编辑按钮，修改姓名或手机号
- 预期：更新成功，列表数据刷新

### 用例 5：删除员工
- 点击删除按钮，确认弹窗点击“确认”
- 预期：删除成功，列表刷新

### 用例 6：未登录访问
- 清除本地 token 后访问 /dashboard
- 预期：自动跳转到 /login
